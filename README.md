# 🎓 AI-Based Smart Study Planner

An AI-powered Smart Study Planner that helps students create personalized study schedules, manage tasks efficiently, and improve productivity through intelligent planning.

## ✨ Features

- 📅 Personalized study timetable generation
- 🤖 AI-based study recommendations
- ⏰ Smart task scheduling and reminders
- 📊 Progress tracking dashboard
- 🎯 Goal setting and achievement monitoring
- 📚 Subject-wise study planning
- 🌙 User-friendly interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- AI-based scheduling logic
