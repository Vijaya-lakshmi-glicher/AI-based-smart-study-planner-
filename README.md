# 📚 AI-Based Smart Study Planner

An AI-powered study planning application that helps students create effective study schedules based on subjects, available time, and learning goals.

## 🚀 Features
- Generate personalized study plans
- Subject-wise schedule management
- Daily study recommendations
- Progress tracking
- User-friendly interface
- Responsive design

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- HTML/CSS

## 📦 Installation

```bash
git clone https://github.com/Vijaya-lakshmi-glicher/AI-based-smart-study-planner-.git
cd AI-based-smart-study-planner-
pip install -r requirements.txt
```

## ▶️ Run the Application

```bash
streamlit run app.py
```
